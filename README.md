##1.プログラムの目的

Androidのホーム画面に日付ウィジェットを表示するためのプログラムです。  
この辺りを参考にさせていただいた。  
http://boco.hp3200.com/beginner/widget01-1.html

##2.動作条件

ECLIPSE Mars.1 Release (4.5.1)  
JAVA1.8で書かれています。  
ターゲットAPI 19。コンパイルAPI 23。  
実機302KCで動作確認済み。  

##3.使い方

Androidアプリは階層があるので、うまく配置してビルド。  
ホーム画面をタップ長押しして、ウィジェット一覧から選択。
